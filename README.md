# WP Social Links

Version: 1.0

## Usage

```php
$social_links_options = get_option( 'cbwp_social_links' );

$facebook = $social_links_options['facebook']; // Facebook
$twitter = $social_links_options['twitter']; // Twitter
$instagram = $social_links_options['instagram']; // Instagram
$youtube = $social_links_options['youtube']; // Youtube
$linkedin = $social_links_options['linkedin']; // LinkedIn
$whatsapp = $social_links_options['whatsapp']; // Whatsapp
```
